## 5. 企业 Agent 对接前置清单

企业 agent 接入前建议先确认：

1. agent 使用数据库只读账号，还是调用平台 REST/OpenAPI。
2. agent 是否直接读取 `ProjectAssetView`、`FileAssetView`、`ModelAssetView`、`AuditEventView`。
3. agent 是否需要真实 NAS 路径，还是只拿平台授权下载/打开入口。
4. agent 是否需要平台事件流做增量同步。
5. agent 是否需要触发扫描、checksum 或提交数据治理建议。
6. agent 是否需要读取文件正文；如果需要，需另行确认文件预览/正文抽取方案。
7. 非标准目录治理建议如何保存：建议单独设计“治理建议表/审核表”，不要直接写正式资产库。

## 6. 建议的下一步

建议下一步先做两件事：

1. 将本报告交给业务负责人核对 16 个已接管项目的编码和名称。
2. 与企业 agent 团队确认数据读取方式和非标准资料治理流程。

在业务负责人确认前，不建议继续导入 `98`、`95`、`99` 或投标/参考目录。
