## 6. 发现的问题

本轮没有发现阻塞问题，但暴露出一个产品体验问题：

- 当前扫描接口是同步执行，会先递归遍历目录再过滤白名单扩展名。
- `93` 项目白名单资产虽然可控，但目录整体文件多，扫描耗时约 `3 分 25 秒`。
- 如果后续继续扩大试点，应优先优化扫描为异步进度可见、可中断、可续扫、可跳过低价值目录。
