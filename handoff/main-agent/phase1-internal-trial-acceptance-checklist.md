# 一期内部试运行验收清单

更新时间：2026-05-13

## 验收结论口径

本清单用于公司内部试运行验收，不用于二期客户交付验收。

一期目标是：公司内部 BIM/CAD 项目资产能够统一建档、可检索、可追路径、可看质量问题、权限不泄漏，并为后续企业 agent 接入保留稳定数据库读模型。

本阶段不验收：

- 模型轻量化在线预览。
- 构件级解析、构件级搜索、碰撞检查。
- PDF/Office 正文解析。
- 企业 agent 深度联调。
- 10TB 全量 checksum 完成率。
- 客户私有化交付安装包。

本清单现在包含一期文件预览安全外壳验收。注意：这里验收的是“预览状态和入口”，不是验收真实文件在线预览、Office 转换、CAD 转换或 BIM 轻量化。

## 1. 首页与项目下拉

验收页面：

```text
http://localhost:5173/home
```

应通过：

1. 页面可打开。
2. 右上角“当前项目”下拉只出现 18 个项目。
3. 下拉中应包含 2 个样板项目和 16 个真实 NAS 项目。
4. 不应再出现大量 `BIM-PROJ-*`、`CSV-PROJ-*`、`XLSX-*`、`SCANCTRL-*` 测试项目。
5. 切换项目后页面不报错。

阻塞失败：

1. 项目下拉仍出现大量测试项目。
2. 切换真实项目后页面白屏。
3. 登录后因为旧项目失效无法进入系统。

## 2. 资产总览

验收页面：

```text
http://localhost:5173/data-steward/assets
```

应通过：

1. 默认显示真实 NAS 项目。
2. 项目数为 16 个。
3. 页面能看到项目编码、项目名称、阶段、负责人、模型数、容量、最近更新时间、状态。
4. 项目 ID 升序/降序选择可用。
5. 点击或双击项目可以进入项目详情。
6. 搜索项目编码或名称能筛到对应项目。

建议抽查项目：

```text
506 / 93-中建八局国交酒店项目
505 / 101-C塔
512 / 108-福城南产业片区11-20-02宗地
519 / 115-深圳宝安国际机场T2航站区及配套设施工程航站区工程施工总承包2标段
```

阻塞失败：

1. 真实 NAS 项目缺失。
2. 搜索项目找不到已登记项目。
3. 进入详情页失败。
4. 项目容量或文件数明显为 0。

## 3. 项目资产详情

验收页面示例：

```text
http://localhost:5173/data-steward/assets/506
```

应通过：

1. 页面可打开。
2. 文件列表有数据。
3. 文件名、类型、专业、版本、大小、状态、路径等字段可见。
4. 专业字段不应全是 `-`，应显示结构、建筑、电气、暖通、给排水、消防、智能化、综合等中文专业。
5. 点击“复制路径”应成功。
6. 路径应指向 `/Volumes/zyzn/卓羽智能项目/...` 下的真实 NAS 路径。
7. 按质量问题筛选时，缺专业应为空。
8. 点击“预览”应打开文件预览状态弹窗。
9. 预览弹窗应说明当前只做入口和状态判断，不读取文件正文、不执行真实转换。

建议抽查：

```text
项目 506：文件总数应为 5912。
项目 512：专业字段应已补齐，缺专业筛选为空。
项目 519：checksum 已完成，缺 checksum 筛选为空。
```

阻塞失败：

1. 文件列表为空但数据库中有文件。
2. 复制路径失败。
3. 路径丢失或不是 NAS 路径。
4. 大量专业字段仍显示为 `-`。
5. 普通用户能看到无权限项目文件。
6. 预览入口误导用户以为已经完成真实 BIM/Office/CAD 在线预览。

## 4. 扫描任务

验收页面：

```text
http://localhost:5173/data-steward/scans
```

应通过：

1. 页面可打开。
2. 能看到历史扫描任务。
3. 成功任务进度应显示 100%，不应显示 0% 但状态成功。
4. 项目字段不应空白；全局任务可显示“全局扫描 / 未绑定具体项目”。

2. 项目字段大量空白且无法解释。
3. 打开扫描报告报错。

## 5. 数据质量

验收页面：

```text
http://localhost:5173/data-steward/quality
```

应通过：

1. 页面可打开。
2. 缺专业应为 0。
3. 缺路径应为 0。
4. 零大小文件应为 0。
5. 缺 checksum 可以存在，当前约 39,498，不阻塞。
6. 非标准待治理应为 11。
7. 点击风险项目可跳转到项目详情并筛选对应质量问题。

阻塞失败：

1. 缺路径不为 0。
2. 缺专业不为 0。
3. 质量页跳转到项目详情失败。
4. 风险指标明显无法解释。

## 6. 非标准资料治理

验收页面：

```text
http://localhost:5173/data-steward/nonstandard-directories
```

应通过：

1. 页面可打开。
2. 当前应有 11 条待治理目录。
3. 能看到重复编号、投标/参考、未知编码、用户暂缓等类型。
4. `95`、`98`、`99` 等暂缓项目应留在治理区。

阻塞失败：

1. 非标准目录进入正式资产库。
2. `95`、`98`、`99` 被误导入正式项目。
3. 治理记录无法查看。

## 7. 权限抽查

使用普通项目用户登录：

```text
delivery.engineer / Engineer@123
```

应通过：

1. 普通用户不能看到未授权真实 NAS 项目。
2. 普通用户不能访问 `101-C塔`、`93-中建八局国交酒店项目` 等管理员项目文件。
3. 普通用户不能看到或操作不属于自己的扫描任务、审核候选、删除申请。

阻塞失败：

1. 普通用户能看到管理员项目文件。
2. 普通用户能进入真实 NAS 项目详情。
3. 普通用户能下载或复制无权限文件路径。

## 8. 企业 Agent 接入准备

本阶段只验收“准备好”，不验收 Hermes 联调。

应通过：

1. 本机只读账号 `hermes_agent_ro` 可读取稳定 View。
2. 稳定 View 包含：

```text
ProjectAssetView
FileAssetView
ModelAssetView
AuditEventView
```

3. 只读账号不能读取业务底表。
4. 事件同步策略为后续 agent 按 `AuditEventView.event_id` 拉取。

已通过脚本：

```text
scripts/dev/check-agent-db2-contract.sh
```

阻塞失败：

1. 只读账号能读业务底表。
2. 四个稳定 View 缺失。
3. 事件流无稳定游标。

## 9. 文件预览安全外壳

验收页面：

```text
http://localhost:5173/data-steward/assets/506
```

应通过：

1. 文件表格中有“预览”入口。
2. 点击后能打开“文件预览状态”弹窗。
3. 弹窗能看到文件 ID、文件名、项目、文件类型、预览状态、预览方式、转换状态。
4. PDF/图片类显示为可接入浏览器原生预览。
5. BIM 文件显示为需要后续 BIM 轻量化转换。
6. CAD 文件显示为需要后续 CAD 转换。
7. Office 文件显示为需要后续 Office 转换。
8. 弹窗文案明确当前阶段不读取文件正文、不执行真实转换。
9. 预览接口不应返回真实 `storagePath`。
10. 页面不应横向撑爆或卡死。

阻塞失败：

1. 无权限用户能查看未授权文件的预览状态。
2. 预览接口向无权限用户泄露真实路径。
3. 弹窗文案暗示已经完成真实文件在线预览、模型轻量化或正文解析。
4. 点击预览导致页面卡死、白屏或横向无限扩张。

问题分级：

```text
P0：权限泄漏、找不到真实文件、正式资产污染、页面主链路无法打开。
P1：路径复制失败、关键字段错误、质量指标明显错误、重要筛选不可用。
P2：显示文案、排序、布局、非关键提示问题。
```

## 最终通过标准

一期内部试运行可以通过的条件：

1. 管理员能看到 16 个真实 NAS 项目。
2. 真实项目文件可检索、可查看元数据、可复制路径。
3. 专业、路径、大小等关键字段不缺失。
4. 非标准目录留在治理区，不污染正式库。
5. 普通用户不能越权访问真实项目。
6. 数据质量页的问题能解释。
7. 企业 agent 的稳定数据库读模型已准备好。
8. 文件预览安全外壳能正确表达预览状态和后续转换要求。

只要以上条件满足，即使仍有 checksum 待补，也可以认定一期内部试运行具备验收基础。
